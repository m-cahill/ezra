### Registry Pattern

Uses lazy import pattern to avoid importing heavy ML modules at registry import time:

```python
_PLUGIN_REGISTRY: dict[str, str] = {
    "easyocr": "ezra.plugins.easyocr_plugin:EasyOCRPlugin",
}
```

Plugins are resolved dynamically when `get_plugin()` is called.
